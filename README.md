# AI-Driven Personalized Learning and Behavioral Assessment Platform

An AI-driven platform that performs adaptive aptitude assessment based on student performance and behavioral analytics, using a lightweight rule-based engine.

## Features
- **General Aptitude Test**: Evaluates student baseline knowledge.
- **Adaptive Engine**: Dynamically adjusts question difficulty based on accuracy and response time.
- **Analytics Dashboard**: Visualizes performance metrics (accuracy, response time, difficulty distribution).
- **Role-Based Access**: Separate portals for Students and Administrators.

## Tech Stack
- **Frontend**: React (Vite), Tailwind CSS, Framer Motion, Recharts.
- **Backend**: Node.js, Express.js.
- **Database**: MySQL (XAMPP compatible).
